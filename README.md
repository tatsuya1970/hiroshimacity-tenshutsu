# 広島市 転入超過・転出超過フロー

広島市と日本の各都道府県との間の「転入 − 転出」の差し引き（純流動・超過分）を、ドットの流れで可視化したシングルページHTMLです。

## 概要

- 中国地方を中心に、大阪・東京までを含む地図上で、広島市と47都道府県の純流動を表現
- 赤いドット（→広島市方向）: その県からの **転入超過**
- 青いドット（広島市→方向）: その県への **転出超過**
- 出現頻度 = 超過人数に比例（恣意的な増減なし）

## データソース

**広島市統計書（令和6年版）** 第3章「B 人口」より：

| 表番号 | 表名 | 用途 |
|---|---|---|
| B-6 | 前住地（区，都道府県）別転入者数 | 県別 転入人口 |
| B-7 | 転出先（区，都道府県）別転出者数 | 県別 転出人口 |

- 対象年: **令和5年（2023年）**
- 対象者: **日本人のみ**（原典の注書きに基づく）
- 入手元: https://www.city.hiroshima.lg.jp/shisei/toukei/1006109/1027848/1037846.html

## 可視化ルール

| パラメータ | 値 | 備考 |
|---|---|---|
| 1ドット | **= 10人** | 量子化単位 |
| 出現頻度 | 超過人数に**線形比例** | 1755人/年 ≒ 2.93個/秒 |
| ドットの大きさ | **固定（半径6.8px）** | 流量は表さない |
| ドットの速さ | **固定（約1.2秒/パス）** | 演出のみ、量とは無関係 |
| 時間圧縮 | **1年 = 1分** | 約52万倍速 |

## 表示

`index.html` をブラウザで開くだけで動きます。外部依存ゼロ（CDN不要・オフラインOK）の単一ファイル構成です。

```sh
# ローカルプレビュー（任意の方法）
open index.html
# あるいは Python 簡易サーバー
python -m http.server 8000
```

## データ再生成

データを差し替えたり地図境界を再簡略化する場合：

```sh
# 1. 広島市統計書 xlsx を data/hiroshima_b.xlsx に配置
# 2. xlsx を解析して TSV 出力
node data/parse_xlsx.js sheet7.xml > data/b06_raw.tsv
node data/parse_xlsx.js sheet8.xml > data/b07_raw.tsv

# 3. 県別フローを JSON で抽出
node data/extract_flows.js > data/flows.json

# 4. 県境 GeoJSON を簡略化（オプション）
node data/simplify_geo.js 0.02 0.005

# 5. 単一HTMLをビルド
node data/build_html.js
# → ../index.html を生成
```

## ファイル構成

```
.
├── index.html                       # 成果物（単一HTML・自己完結）
├── README.md
├── .gitignore
└── data/
    ├── hiroshima_b.xlsx             # 広島市統計書 B 人口 原典
    ├── parse_xlsx.js                # xlsx → TSV パーサ（依存ゼロ）
    ├── extract_flows.js             # TSV → flows.json
    ├── simplify_geo.js              # GeoJSON 簡略化（Douglas-Peucker）
    ├── flows.json                   # 抽出済み県別フロー
    ├── japan_simplified.geojson     # 簡略化済み都道府県境界（約60KB）
    ├── template.html                # ビルド用テンプレート
    └── build_html.js                # template + data → index.html
```

## 制限事項

- 表示は **転入と転出の差し引き（純流動）** です。グロスの移動量ではありません
- 県内市町村別の内訳は原資料に含まれないため、広島県内の他市町（福山・呉・東広島など）からの流動はすべて **「広島県」** に丸めて集計
- 曲線（ベジエ）と地図投影は表現上のもので、実際の移動経路を示しません
- 外国人は含まれません
